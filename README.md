# Code Challenge Platform

A code challenge platform built with React, Node.js, and Flask.

## Technologies Used:

### Frontend (React):
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Jest**
- **CodeMirror**: A versatile text editor implemented in JavaScript.

### Backend (Node.js and Flask):
- **Node.js**
- **Flask**
- **Unittest**
- **vm2**
